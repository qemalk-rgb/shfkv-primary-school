# Shkolla Fillore – Full Editable Website

Ky version është rindërtuar që përmbajtja kryesore të redaktohet nga **admin.html**, pa hyrë në kod.

## Si përdoret
1. Hap `index.html` për faqen publike.
2. Hap `admin.html` për panelin.
3. Fjalëkalimi fillestar: `admin123`.
4. Redakto dhe ruaj përmbajtjen.

## Çfarë mund të redaktohet nga admin paneli
- Emri i shkollës në 4 gjuhë: Shqip, Anglisht, Turqisht, Maqedonisht
- Logo tekst ose logo foto
- Telefon, email, adresë, orar, rrjete sociale
- Ballina: foto kryesore, titull, tekst, statistika, partnerë
- Rreth nesh: teksti dhe timeline/historia
- Lajme: shto/redakto/fshi lajme me foto
- Mësimdhënës: shto/redakto/fshi mësimdhënës me foto
- Galeri: shto/redakto/fshi fotografi
- Dokumente: shto/redakto/fshi dokumente

## Shënim i rëndësishëm
Ky është version **HTML/CSS/JavaScript pa databazë**. Ndryshimet ruhen në `localStorage`, pra në browserin ku punon administratori. Për faqe zyrtare ku ndryshimet duhet t’i shohin të gjithë automatikisht nga çdo pajisje, duhet backend/databazë ose CMS real.
