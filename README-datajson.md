# Version me data.json

Ky version përdor një file të vetëm `data.json` për përmbajtjen e faqes.

## Si punohet në GitHub Pages
1. Ngarko në GitHub: `admin.js`, `lang.js`, `style.css` dhe `data.json`.
2. Hape `admin.html` në faqen tënde.
3. Bëj ndryshimet dhe kliko **Ruaj**.
4. Kliko **Eksporto data.json**.
5. Në GitHub hap file-in `data.json` dhe zëvendësoje me file-in e ri të eksportuar.
6. Bëj **Commit changes**. Pas pak minutash faqja publike do të shfaqë ndryshimet.

## Shënim
Në GitHub Pages admini nuk mund ta ndryshojë automatikisht repository-n. Prandaj duhet zëvendësuar manualisht vetëm `data.json`.
